      DESENVOLVIMENTO

      AWS CodeStar

      Visão geral:

      • Descrição: O AWS CodeStar permite desenvolver, criar e implantar rapidamente aplicativos na AWS. Ele oferece uma interface unificada para gerenciar atividades de desenvolvimento de software em um só lugar.
      • Características:
        • Configuração rápida da cadeia de ferramentas de entrega contínua.
        • Facilita a colaboração segura da equipe, gerenciando acessos e adicionando proprietários, contribuidores e visualizadores aos projetos.
        • Suporte a uma variedade de linguagens de programação e modelos de projeto para desenvolver em Amazon EC2, AWS Lambda e AWS Elastic Beanstalk.

      Link: AWS CodeStar Características

      AWS CodeCommit

      Visão geral:

      • Descrição: AWS CodeCommit é um serviço de controle de origem totalmente gerenciado que hospeda repositórios seguros baseados em Git.
      • Características:
        • Facilita a colaboração em código em um ecossistema seguro e altamente escalável.
        • Elimina a necessidade de operar seu próprio sistema de controle de origem.
        • Funciona perfeitamente com ferramentas Git existentes.

      Link: AWS CodeCommit Características

      AWS CodeBuild

      Visão geral:

      • Descrição: AWS CodeBuild é um serviço de integração contínua totalmente gerenciado que compila código-fonte, executa testes e produz pacotes de software prontos para implantação.
      • Características:
        • Elimina a necessidade de provisionar, gerenciar e escalar seus próprios servidores de compilação.
        • Escala continuamente e processa várias compilações simultaneamente.
        • Permite começar rapidamente com ambientes de compilação pré-empacotados ou criar ambientes personalizados.

      Link: AWS CodeBuild Características

      AWS CodeDeploy

      Visão geral:

      • Descrição: AWS CodeDeploy é um serviço de implantação totalmente gerenciado que automatiza implantações de software em uma variedade de serviços de computação, como Amazon EC2, AWS Lambda e servidores locais.
      • Características:
        • Facilita a rápida liberação de novos recursos.
        • Evita tempo de inatividade durante a implantação de aplicativos.
        • Automatiza implantações de software, eliminando operações manuais propensas a erros.
        • Escala para atender às necessidades de implantação, desde uma única função Lambda até milhares de instâncias EC2.

      Link: AWS CodeDeploy Características

      AWS CodePipeline

      Visão geral:

      • Descrição: AWS CodePipeline é um serviço de entrega contínua totalmente gerenciado que ajuda a automatizar os pipelines de lançamento para atualizações rápidas e confiáveis de aplicativos e infraestrutura.
      • Características:
        • Automatiza as fases de construção, teste e implantação sempre que há uma alteração de código.
        • Facilita a entrega rápida e confiável de recursos e atualizações.
        • Integra facilmente com serviços de terceiros, como o GitHub, ou com plugins personalizados.

      Link: AWS CodePipeline Características

      AWS X-Ray

      Visão geral:

      • Descrição: AWS X-Ray ajuda os desenvolvedores a analisar e depurar aplicativos distribuídos em produção, como aqueles construídos usando uma arquitetura de microsserviços.
      • Características:
        • Permite entender o desempenho do aplicativo e seus serviços subjacentes.
        • Identifica e soluciona a causa raiz de problemas de desempenho e erros.
        • Oferece uma visualização de ponta a ponta de solicitações enquanto viajam pelo aplicativo.

      Link: AWS X-Ray Características

      AWS Service Catalog

      Visão geral:

      • Descrição: AWS Service Catalog permite que as organizações criem e gerenciem catálogos de serviços aprovados para ajudar os desenvolvedores a implantar serviços de forma rápida e compatível com políticas.
      • Características:
        • Oferece catálogos personalizados para serviços aprovados.
        • Facilita a implantação rápida e segura de serviços.
        • Mantém o controle sobre recursos implantados e sua conformidade.

      Link: AWS Service Catalog Características
