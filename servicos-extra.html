      SERVIÇOS EXTRA

      COMPUTAÇÃO

      Amazon Elastic Container Service for Kubernetes (EKS)

      • Amazon Elastic Container Service for Kubernetes (EKS) é um serviço gerenciado Kubernetes que facilita a execução do Kubernetes na AWS sem a necessidade de instalar, operar e manter seu próprio plano de controle do Kubernetes.
      • Certificado como conformante ao Kubernetes, o EKS permite que aplicativos existentes em execução no Kubernetes padrão sejam compatíveis com o Amazon EKS.
      • O EKS gerencia automaticamente a disponibilidade e escalabilidade dos nós do plano de controle do Kubernetes, responsáveis por iniciar e parar contêineres, agendar contêineres em máquinas virtuais, armazenar dados do cluster e outras tarefas.
      • O EKS detecta e substitui automaticamente nós do plano de controle não saudáveis para cada cluster.
      • Geralmente disponível, mas atualmente disponível apenas em regiões limitadas.

      BANCO DE DADOS

      Amazon Neptune

      • Amazon Neptune é um serviço de banco de dados de gráficos totalmente gerenciado, rápido e confiável, que facilita a construção e execução de aplicativos que trabalham com conjuntos de dados altamente conectados.
      • Com o Amazon Neptune, é possível criar aplicativos de gráficos sofisticados e interativos que podem consultar bilhões de relações em milissegundos.
      • Consultas SQL para dados altamente conectados são complexas e difíceis de ajustar para o desempenho. Em vez disso, o Amazon Neptune permite o uso das linguagens de consulta de gráficos populares Apache TinkerPop Gremlin e SPARQL da W3C para executar consultas poderosas que são fáceis de escrever e têm bom desempenho em dados conectados.

      MIGRAÇÃO

      AWS Migration Hub

      • AWS Migration Hub fornece um local único para acompanhar o progresso das migrações de aplicativos em várias soluções da AWS e de parceiros.
      • Usar o Migration Hub permite escolher as ferramentas de migração da AWS e do parceiro que melhor atendem às suas necessidades, fornecendo visibilidade sobre o status das migrações em seu portfólio de aplicativos.
      • Por exemplo, você pode usar o AWS Database Migration Service, AWS Server Migration Service e ferramentas de migração de parceiros, como ATADATA ATAmotion, CloudEndure Live Migration ou RiverMeadow Server Migration SaaS para migrar um aplicativo composto por um banco de dados, servidores web virtualizados e um servidor bare metal.
      • Usando o Migration Hub, é possível visualizar o progresso da migração de todos os recursos no aplicativo.

      SERVIÇO DE MIGRAÇÃO DE BANCO DE DADOS

      • AWS Database Migration Service ajuda a migrar bancos de dados para a AWS de forma rápida e segura.
      • O banco de dados de origem permanece totalmente operacional durante a migração, minimizando o tempo de inatividade para aplicativos que dependem do banco de dados.
      • O AWS Database Migration Service pode migrar seus dados de e para a maioria dos bancos de dados amplamente utilizados comerciais e de código aberto.
      • O serviço oferece suporte a migrações homogêneas, como Oracle para Oracle, bem como migrações heterogêneas entre plataformas de banco de dados diferentes, como Oracle ou Microsoft SQL Server para Amazon Aurora.
      • Com o AWS Database Migration Service, é possível replicar continuamente seus dados com alta disponibilidade e consolidar bancos de dados em um data warehouse de petabytes, transmitindo dados para o Amazon Redshift e o Amazon S3.

      Mais informações

      SERVIÇO DE MIGRAÇÃO DE SERVIDORES

      • AWS Server Migration Service (SMS) é um serviço sem agente que facilita e acelera a migração de milhares de cargas de trabalho locais para a AWS.
      • O AWS SMS permite automatizar, agendar e rastrear replicação incremental de volumes de servidor em tempo real, facilitando a coordenação de migrações de servidores em grande escala.

      REDE & ENTREGA DE CONTEÚDO

      Amazon API Gateway

      • Amazon API Gateway é um serviço totalmente gerenciado que facilita para os desenvolvedores criar, publicar, manter, monitorar e proteger APIs em qualquer escala.
      • Com alguns cliques no Console de Gerenciamento da AWS, é possível criar uma API que atua como uma "porta de entrada" para que aplicativos acessem dados, lógica de negócios ou funcionalidades de seus serviços back-end.
      • Serviços back-end podem incluir Amazon Elastic Compute Cloud (Amazon EC2), código em execução no AWS Lambda ou qualquer aplicativo da web.

      SERVIÇOS GERENCIADOS

      AWS Managed Services

      O AWS Managed Services oferece a gestão contínua da sua infraestrutura AWS para que você possa se concentrar em suas aplicações.

      • Implementa as melhores práticas para manter sua infraestrutura.
      • Reduz sobrecarga operacional e risco.
      • Automação de atividades comuns, como solicitações de alteração, monitoramento, gerenciamento de patches, segurança e serviços de backup.
      • Serviços de ciclo de vida completo para provisionar, executar e dar suporte à sua infraestrutura.

      Mais informações

      ANÁLISE

      Amazon Athena

      Amazon Athena é um serviço de consulta interativo que facilita a análise de dados no Amazon S3 usando SQL padrão.

      • Serverless, sem infraestrutura para gerenciar.
      • Pagamento apenas pelas consultas executadas.
      • Escalabilidade automática para consultas rápidas, mesmo com grandes conjuntos de dados e consultas complexas.

      Link: Amazon Athena Características

      Amazon EMR

      Amazon Elastic MapReduce (EMR) fornece um framework Hadoop gerenciado para processar grandes volumes de dados de maneira fácil, rápida e econômica.

      • Suporta frameworks distribuídos como Apache Spark, HBase, Presto e Flink.
      • Integração com armazenamento de dados AWS, como Amazon S3 e Amazon DynamoDB.
      • Manuseio seguro e confiável de casos de uso de big data, como análise de logs, indexação da web, transformações de dados (ETL), aprendizado de máquina, análise financeira, simulação científica e bioinformática.

      Link: Amazon EMR Características

      Amazon CloudSearch

      Amazon CloudSearch é um serviço gerenciado na AWS que simplifica e torna acessível a configuração, gerenciamento e escalabilidade de uma solução de pesquisa para sites ou aplicativos.

      • Suporta 34 idiomas e recursos populares de pesquisa, como destaque, autocompletar e pesquisa geoespacial.

      Link: Amazon CloudSearch

      Amazon Elasticsearch

      • Descrição: Amazon Elasticsearch Service é um serviço totalmente gerenciado que facilita implantar, proteger, operar e dimensionar o Elasticsearch para busca, análise e visualização de dados em tempo real.
      • Características:
        • APIs fáceis de usar e capacidades de análise em tempo real.
        • Casos de uso, como análise de logs, pesquisa de texto completo, monitoramento de aplicativos e análise de fluxo de cliques.
        • Disponibilidade, escalabilidade e segurança corporativa.

      Link: Amazon Elasticsearch Service Características

      Amazon Kinesis

      • Descrição: Amazon Kinesis facilita a coleta, processamento e análise de dados em tempo real, permitindo insights oportunos e reações rápidas a novas informações.
      • Types:
        • Kinesis Video Streams: Transmissão segura de vídeo para análise e aprendizado de máquina.
        • Kinesis Data Streams: Construção de aplicativos personalizados para processar ou analisar dados em streaming.
        • Kinesis Data Firehose: Carregamento fácil de dados em fluxo contínuo em armazenamento e ferramentas analíticas.
        • Amazon Kinesis Data Analytics: Processamento e análise simples de dados em streaming.

      Link: Amazon Kinesis

      AWS Data Pipeline

      • Descrição: AWS Data Pipeline é um serviço da web que ajuda a processar e mover dados de maneira confiável entre diferentes serviços de computação e armazenamento AWS, bem como fontes de dados locais, em intervalos especificados.
      • Características:
        • Acesso regular aos dados onde estão armazenados.
        • Transformação e processamento em escala.
        • Transferência eficiente dos resultados para serviços AWS como Amazon S3, Amazon RDS, Amazon DynamoDB e Amazon EMR.

      Link: AWS Data Pipeline

      AWS Glue

      • Descrição: AWS Glue é um serviço totalmente gerenciado de extração, transformação e carga (ETL) que facilita a preparação e o carregamento de dados para análise.
      • Características:
        • Criação e execução de trabalhos ETL com alguns cliques no Console de Gerenciamento da AWS.
        • Descoberta e catalogação automática de dados armazenados na AWS.
        • Geração automática de código para executar transformações e carregamento de dados.

      Link: AWS Glue Características

      MEDIA SERVICES

      Amazon Elastic Transcoder

      Amazon Elastic Transcoder é transcodificação de mídia na nuvem, projetada para ser altamente escalável, fácil de usar e uma maneira econômica para desenvolvedores e empresas converterem arquivos de mídia de seu formato de origem para versões que reproduzirão em dispositivos como smartphones, tablets e PCs.

      Link: Amazon Elastic Transcoder

      MOBILE SERVICES

      AWS AppSync

      AWS AppSync facilita a construção de aplicativos móveis e baseados em navegador orientados por dados, oferecendo experiências colaborativas e responsivas, mantendo os dados atualizados quando os dispositivos estão conectados, permitindo que o aplicativo use dados locais quando estiver offline e sincronizando os dados quando os dispositivos se reconectam.

      • Usa a linguagem de consulta GraphQL padrão para solicitar, alterar e assinar dados.

      Link: AWS AppSync Product Details

      AWS Device Farm

      AWS Device Farm é um serviço de teste de aplicativos que permite testar e interagir com seus aplicativos Android, iOS e web em vários dispositivos ao mesmo tempo, reproduzir problemas em um dispositivo em tempo real e visualizar vídeo, capturas de tela, logs e dados de desempenho para identificar e corrigir problemas.

      Link: AWS Device Farm

      End User Computing

      Amazon WorkSpaces

      Amazon WorkSpaces é um serviço de desktop na nuvem gerenciado e seguro. É possível provisionar desktops Windows ou Linux em apenas alguns minutos e dimensionar rapidamente para fornecer milhares de desktops a trabalhadores em todo o mundo.

      • Oferece uma experiência de desktop virtual segura e gerenciada na nuvem.
      • Não requer preocupações com a complexidade da infraestrutura, pois é totalmente gerenciado.

      Link: Amazon WorkSpaces Características

      AWS AppStream

      Serviço totalmente gerenciado de streaming de aplicativos não persistentes.

      • Alternativa: Alternativa a produtos populares, como Citrix XenApp.

      Link: AWS AppStream 2.0 Características

      AWS WorkLink

      Oferece acesso seguro de um clique a sites e aplicativos da web internos usando navegadores de telefones celulares. Não requer cliente VPN ou aplicativo.

      Link: AWS WorkLink Características

      AWS WorkDocs

      Serviço totalmente gerenciado de criação, armazenamento e colaboração segura de conteúdo. Permite criar, editar e compartilhar conteúdo armazenado centralmente na AWS.

      Link: AWS WorkDocs Características

      INTERNET DAS COISAS (IoT)

      AWS IoT Core

      Descreve a rede de objetos físicos incorporados com sensores ou software. Cada dispositivo IoT pode se comunicar e trocar dados com outros dispositivos e sistemas.

      • Casos de Uso:
        • Automação residencial inteligente.
        • Saúde inteligente.
        • Manufatura.
        • Agricultura.
      • Características:
        • Conecta dispositivos IoT à nuvem AWS sem a necessidade de provisionar ou gerenciar servidores.
        • Pode suportar bilhões de dispositivos e trilhões de mensagens.

      Link: AWS IoT Core Características
